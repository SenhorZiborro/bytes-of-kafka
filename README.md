# Bytes of Kafka

Uma coleção de notas rápidas, conceitos fundamentais e configurações essenciais sobre o Apache Kafka. O objetivo aqui é simplificar o complexo e focar no que realmente importa para o dia a dia de quem opera ou desenvolve com Kafka. Para quem está começando no mundo de Kafka não se esqueça de executar as recomendações de cada laboratório, só assim vai ser possível fixar o conhecimento.
